# Project Mirage

Project Mirage is a Netflix-style cloud platform simulation designed to demonstrate modern DevOps practices.

The platform is built using a multi-service architecture and focuses on:

- containerized microservices
- CI/CD automation
- infrastructure as code
- Kubernetes orchestration
- monitoring and observability

## Platform Services

auth-service  
Handles authentication and token validation.

catalog-service  
Provides movie and show metadata.

recommendation-service  
Provides trending and recommended content.

api-gateway  
Acts as the unified entry point for backend services.

frontend  
User-facing interface for interacting with the platform.
